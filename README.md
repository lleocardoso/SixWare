<h2 align="left">🌱 SixWare: Gestão Hídrica de Precisão </h2>


<h3 align="left">Sobre o Projeto</h3>
<hr>

<p align="left">O SixWare é um protótipo de software para automação e monitoramento de solos. O foco do sistema é processar dados coletados por sensores IoT (umidade e temperatura) e aplicar regras de negócio para acionar a irrigação de forma inteligente, além de registrar alertas e inspeções de pragas.<br><br>Este projeto está sendo desenvolvido como parte da matéria de Projeto Interdisciplinar na graduação em Sistemas de Informação, focando na aplicação prática de engenharia de software no agronegócio.</p>

###

<h3 align="left">Tecnologias Utilizadas</h3>
<hr>

<p align="left"> <li> <strong> Java </strong>: Lógica de negócio, motor de regras e estrutura de dados.</li> <br> <li> <strong> POO (Programação Orientada a Objetos)</strong>: Herança, encapsulamento e polimorfismo aplicados na modelagem do domínio agrícola.</li> <br> <li> <strong>Modelagem de negócios </strong>: Modelagem UML, Casos de Uso e atividades.</li> <br> <li> <strong>Banco de dados </strong>: Modelagem Conceitual e Lógica do banco de dados.</li></p>

###

<h3 align="left">Arquitetura e Modelagem</h3>

<hr>

<p align="left"><li> <strong> Modelos (src/model) </strong>: Representação das entidades reais como Solo, Zona, SensorTemperatura, ControladorIrrigacao, entre outros.</li> <br> <li> <strong> Tipificações (src/enums) </strong>: Padronização de estados e tipos do sistema, garantindo integridade dos dados (ex: StatusValvula, TipoPraga).</li></p>
